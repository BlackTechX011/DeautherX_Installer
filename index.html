How To Flash DeautherX firmware to Esp8266 based boad
  • Select your esp8266 boad
  • Select display type
  • Connect your ESP8266 board via USB
  • Click Connect and Select the serial port of your ESP
  • on Device Dashboard Click on install
How To open web based serial montor
  • Select your esp8266 boad
  • Select display type (it is not important to select correct display type.it is just because without it program will not run)
  • Connect your ESP8266 board via USB
  • Click Connect and Select the serial port of your ESP
  • on Device Dashboard Click on Logs and console
